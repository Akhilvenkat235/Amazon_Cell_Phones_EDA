**Project Overview
**
This project analyzes consumer preferences and pricing trends in the Amazon cell phones market using the Amazon Cell Phones Cleaned Scraped dataset from Kaggle.

The goal is to identify how factors such as price, brand, ratings, number of reviews, and customer feedback influence consumer purchasing behavior. The project applies exploratory data analysis, statistical modeling, sentiment analysis, and machine learning techniques to generate actionable insights for e-commerce sellers.

**Problem Statement**
Amazon cell phone sellers face strong competition and must understand which product attributes drive customer decisions. This project investigates the relationship between pricing, reviews, ratings, and brand performance to help sellers optimize product listings, pricing strategies, and customer satisfaction.

**Objectives**
Identify key factors influencing cell phone purchase decisions on Amazon.
Analyze pricing trends across brands and product categories.
Examine the relationship between price, ratings, and review volume.
Compare brand performance based on customer satisfaction and pricing.
Build predictive models to estimate consumer satisfaction.
Provide data-driven recommendations for Amazon sellers.

**Dataset**

Source: Kaggle
Dataset Link: https://www.kaggle.com/datasets/michaelmatta0/amazon-cell-phones-cleaned-scraped-data

1. Data Cleaning and Preprocessing

3. Statistical Analysis
Performed descriptive statistics
Created correlation analysis
Studied relationships between price, ratings, and reviews
4. Sentiment Analysis
Analyzed customer review text
Classified sentiment as positive, negative, or neutral
Compared sentiment scores with product ratings
5. Machine Learning
Built regression models to predict product ratings
Applied clustering techniques to group similar products
Identified product segments based on price, brand, ratings, and reviews
